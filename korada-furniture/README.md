# Korada Furniture - Modern E-commerce Website

A modern, responsive e-commerce website for Korada Furniture Co. Pvt. Ltd., built with Next.js 15, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Fast Performance**: Built with Next.js 15 for optimal performance
- **SEO Optimized**: Proper meta tags and structured data
- **Interactive UI**: Smooth animations and micro-interactions using Framer Motion
- **Accessible**: Built with accessibility best practices

## 📱 Pages

- **Homepage**: Hero section, product showcase, and features
- **Products**: Comprehensive product catalog with categories
- **About**: Company story, values, and team information
- **Contact**: Contact form and store information
- **Catalogue**: Downloadable product catalogues
- **Dealers**: Dealer application and partnership information

## 🛠️ Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Custom components with Headless UI

## 🚀 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## 🎨 Design Features

- **Color Scheme**: Amber and orange gradient theme
- **Typography**: Inter font for modern readability
- **Components**: Reusable UI components
- **Animations**: Smooth page transitions and hover effects
- **Layout**: Grid-based responsive layout

## 📱 Responsive Design

- Mobile-first approach
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px)
- Optimized for all screen sizes

## 🔧 Customization

The website is built with modularity in mind. You can easily:

- Update colors in `tailwind.config.js`
- Modify components in `src/components/`
- Add new pages in `src/app/`
- Update content in respective page files
